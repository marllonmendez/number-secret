## Sobre

Este projeto foi concebido como parte da trilha de lógica de programação na plataforma Alura em parceria com a Oracle, um desafio que visa criar um jogo para aprimorar habilidades lógicas e estilísticas, com foco na abordagem Mobile First.
O [Oracle Next Education](https://www.oracle.com/br/education/oracle-next-education/) é um programa de educação e empregabilidade com objetivo social de capacitar pessoas em tecnologia e conectá-las com o mercado de trabalho por meio de empresas parceiras.

## Funcionalidades

- Adivinhar um número secreto de 0 até 10
